Every part of this world runs on shorthand nobody stops to expand. This page is a quick reference for the terms used across the guide and the ones you'll hit in college servers and placement drives. Definitions are one line and deliberately plain.

<Mark>If a term on this site or in a placement chat looks like alphabet soup, it's probably here. Skim, don't read.</Mark>

### College and admissions

| Term | What it means |
|------|---------------|
| CSE | Computer Science and Engineering, the core degree this guide is written around. |
| CGPA | Cumulative Grade Point Average, your grades averaged across all semesters. See [CGPA](/college/cgpa). |
| Backlog | A subject you failed and haven't yet cleared. Active backlogs block many placement drives. |
| NAAC | National Assessment and Accreditation Council, which grades institutions on quality. |
| NIRF | National Institutional Ranking Framework, the government's ranking of colleges. |
| NBA | National Board of Accreditation, which accredits specific programs, not whole colleges. |
| AICTE | All India Council for Technical Education, the body regulating technical education. |
| Autonomous college | A college that sets its own syllabus and exams, within a university's affiliation. |
| Deemed university | An institution with university-level independence granted by the government. |
| Lateral entry | Joining a degree directly in the second year, usually after a diploma. See [Admission Paths](/choice/admissions). |
| Management quota | Seats filled by the college outside merit counselling, usually at higher fees. See [Admission Paths](/choice/admissions). |

### Pay and placements

| Term | What it means |
|------|---------------|
| CTC | Cost to Company, the total a company spends on you, larger than your take-home. See [Understanding Pay](/resources/compensation). |
| LPA | Lakhs Per Annum, the unit salaries are quoted in. |
| In-hand | The salary that reaches your bank after deductions, usually 60 to 75 percent of CTC. |
| Base | The fixed part of your salary, before variable pay and bonuses. |
| Variable pay | Salary that depends on ratings or company performance, not guaranteed. |
| EPF / PF | Employee Provident Fund, a retirement contribution counted in CTC but not paid to you monthly. |
| Gratuity | A payout accrued yearly but received only after five continuous years of service. |
| PPO | Pre-Placement Offer, a full-time offer extended after an internship. |
| Bond | A service agreement requiring you to stay a set period or pay a penalty. |
| Service company | A company that builds and maintains software for clients. Hires in bulk. |
| Product company | A company that builds its own product. Usually pays more, filters harder. |
| ROI | Return on Investment, here, whether the fees are worth the outcomes. See [Placements & ROI](/choice/placements). |

### Programming and interviews

| Term | What it means |
|------|---------------|
| DSA | Data Structures and Algorithms, the core of most technical filters. See [DSA](/programming/dsa). |
| OA | Online Assessment, the timed coding test that screens applicants before interviews. |
| SDE | Software Development Engineer, a common title for the roles you're targeting. |
| CP | Competitive Programming, timed contest solving. Optional for placements. |
| System design | Designing how large systems fit together. Relevant for selective and senior roles. |
| HLD / LLD | High-Level and Low-Level Design, the two altitudes of a system design discussion. |
| HR round | The behavioral and fit interview, distinct from technical rounds. |
| Referral | An existing employee putting your application forward, which often skips the resume filter. |
| OS / DBMS / CN | Operating Systems, Database Management Systems, Computer Networks, the core CS subjects interviews test. See [Technical Interviews](/programming/interviews). |
| OOP | Object-Oriented Programming, a common fundamentals topic in interviews. |

### Tools and open source

| Term | What it means |
|------|---------------|
| Git | The version control tool that records your project's history. See [Version Control](/programming/git). |
| Repo | A repository, a single project tracked by git, usually hosted on GitHub. |
| PR | Pull Request, a proposed change to a project that others review before merging. |
| Commit | A saved snapshot of your changes, with a message describing them. |
| IDE | Integrated Development Environment, the editor and tools you write code in. |
| CI | Continuous Integration, automated checks that run on your code when you push. |
| Open source | Software whose code is public and open to contribution. |
| GSoC | Google Summer of Code, a program that pays students to contribute to open source. |

### Higher studies

| Term | What it means |
|------|---------------|
| GATE | Graduate Aptitude Test in Engineering, the exam for MTech admissions and some PSU jobs. |
| MS | Master of Science, a postgraduate degree, often taken abroad. |
| MTech | Master of Technology, the Indian postgraduate engineering degree. |
| SOP | Statement of Purpose, the essay in a graduate application. |
| LOR | Letter of Recommendation, a reference from a professor or manager for admissions. |
| GRE | Graduate Record Examination, a standardized test some foreign universities require. |
