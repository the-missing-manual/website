## v1.3.0 — June 14, 2025

### Reorganized

- **Navigation order** — Changelog moved to the last position in the sidebar. The reading flow is now: Welcome → Choosing a College → The College Journey → Learning to Program → Resources → Wellbeing → Changelog.
- **Learning to Program section reordered** — Pages now follow a logical progression: Choosing a Language → Dev → Clean Code → DSA → Technical Interviews. Previously started with DSA and Interviews before covering the foundational topics they depend on.
- **Choosing a College index** — Added a card linking to the Links Worth Bookmarking page, which existed in the sidebar but was not featured in the section's index.

### Fixed

- Removed a broken link in the After College page that pointed to a `/careers` section which does not exist.
- Removed duplicate "Core CS Subjects" content from the DSA page. That material now lives only in Technical Interviews, where it belongs. The DSA page cross-references it instead.

## v1.2.0 — June 13-14, 2025

### Added
- **Interview Preparation** page under Learning to Program — covers how to approach technical interviews, what to actually study, how to practice, and common mistakes that hurt candidates who are otherwise capable engineers.
- **OG image generation** — each page now generates a social sharing card automatically from its title and description, so links shared on Discord, LinkedIn, and Twitter render properly.

### Infrastructure
- Analytics added via PostHog to understand which sections and pages are actually being read. This will inform which parts need more depth.
- Fixed URL routing issues affecting several internal page links.

---

## v1.1.0 — June 12, 2025

### Added

Five sections launched with full content:

- **Choosing a College** — 10 pages covering how to build a shortlist, evaluate intake size, assess coding culture, understand internship policies, read sub-branch naming (AI/Data Science/Cyber Security), interpret placement data honestly, navigate admission paths, and factor in location.
- **The College Journey** — Year-by-year breakdown: what to focus on before you arrive, in each of the four years, and immediately after you graduate.
- **Learning to Program** — Where to start if you have no prior experience, how to approach DSA, what "working on projects" actually means, clean code fundamentals, and how to choose a programming language.
- **Resources** — How to search effectively (not just Google things), how to use awesome lists, and how to ask questions in a way that gets useful answers.
- **Wellbeing** — Mental and physical health during placement season and through the degree in general.
